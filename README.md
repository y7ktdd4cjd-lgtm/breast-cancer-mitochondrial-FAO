# Breast Cancer Mitochondrial FAO Suppression — Analysis Code

Code supporting: Gomosani A et al. "Convergent suppression of 
nuclear-encoded mitochondrial fatty acid oxidation genes defines 
a pan-subtype signature in breast cancer." BMC Cancer (under review).

## Requirements
Python 3.10 (Google Colab)
pandas, numpy, scipy, scikit-learn, statsmodels, 
matplotlib, seaborn, pydeseq2

## Data
TCGA-BRCA: https://portal.gdc.cancer.gov (version 09-07-2024)
GSE42568 and GSE109169: https://www.ncbi.nlm.nih.gov/geo/
METABRIC: https://www.cbioportal.org/study/summary?id=brca_metabric

## Usage
Open the .ipynb in Google Colab and run cells sequentially.
Input data files should be placed in Google Drive as described 
in Cell 1 of the notebook.
